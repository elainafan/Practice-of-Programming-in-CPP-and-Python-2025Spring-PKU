由于期末内容由算法改为简单Python，因此25年之前的往年题均无意义。

Python通常考察范围为，Python基础(I)(II)(III)(IV)，以及pandas跟bs4，其余课上知识均不在范围内。

就去年的情况来看，期末的12道题中，有6道为Python基础(I)(II)(III)的内容，4道为Python基础(IV)的内容，1道pandas，1道bs4。

建议的复习方式为，在[程序设计实习Mooc]()上完成Python题目，答案见，巩固Python(I)(II)(III)的知识，作者有几道题并未给出答案，其实是没做。同时记忆Python(IV)的知识点并熟练应用，pandas和bs4的知识点可以考场现学，因为记忆量比较大。
